Le DeerFlow Harness est le SDK Python et le socle d'exécution pour construire vos propres systèmes Super Agent.

Si vous souhaitez composer des agents avec des skills, de la mémoire, des outils, des sandboxes et des sous-agents dans votre propre produit ou workflow, c'est la partie de DeerFlow sur laquelle vous allez construire.

## Nom du package

Le nom du package sera :

```bash
pip install deerflow
```

Ce package n'est pas encore disponible publiquement, mais c'est le chemin d'installation que la documentation utilisera une fois publié.

## État actuel

Le package DeerFlow Harness est **à venir**.

Pour l'instant, cette section existe pour établir le point d'entrée du SDK et l'identité du package, pendant que le flux de distribution publique est en cours de finalisation.

## Ce que le Harness vous apportera

Le Harness est conçu pour les développeurs qui souhaitent construire leur propre système d'agents sur le modèle d'exécution de DeerFlow.

Il fournira les fondations pour :

- construire des agents à long horizon,
- composer des capacités d'exécution telles que la mémoire, les outils, les skills et les sous-agents,
- exécuter des agents avec une exécution en sandbox,
- personnaliser le comportement des agents via la configuration et le code, et
- intégrer DeerFlow dans votre propre architecture applicative.
